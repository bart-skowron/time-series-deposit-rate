# Time Series Analysis - modelling interest rates
I try to model American deposit rate using other lending rates.
I went throught the following steps in the exercise:
- Time series cleaning
- Feature engineering (rolling means and lags)
- EDA
- Modelling
  - SARIMAX model
  - Neural Network
- Model comparison
